## Installation Guides

These guides have been created to help you set up your computer and install the necessary Applications and Tools that you will need to code in C# and C++ with SplashKit.

### <span style="display:none;">Raspberry Pi</span>

<Card title="Raspberry Pi" icon="fa-brands:raspberry-pi">
  Follow these steps to build a Raspberry Pi computer and set it up to get started coding.
  
  For this you will need a *Raspberry Pi 4* or later device, as described in the Hardware section.

  <LinkCard
    title="Raspberry Pi Hardware Setup Guide"
    href="/book/appendix/0-installation/2-2-pi-computer/"
    description="Go to 'Build my Raspberry Pi' page"
  />
  <LinkCard
    title="Raspberry Pi Software Setup Guide"
    href="/book/appendix/0-installation/2-3-setup-pi/"
    description="Go to 'Set up my Raspberry Pi' page"
  />
</Card>

### <span style="display:none;">macOS</span>

<Card title="macOS" icon="fa-brands:apple">
  Follow these steps to set up your Mac computer to get started coding.

  For this you will need a *macOS* computer running OSX 13 (macOS Ventura) or later.
  
  Note: *If you are running an earlier version, make sure to read the [Using older OSX version](https://splashkit.io/troubleshoot/macos/mac-issue-8/) troubleshooting guide once you have finished the setup steps, or if you have any issues.*

  <LinkCard
    title="macOS Software Setup Guide"
    href="/book/appendix/0-installation/2-1-setup-macos/"
    description="Go to 'Set up my macOS' page"
  />
</Card>

### <span style="display:none;">Windows</span>

<Card title="Windows" icon="fa-brands:windows">
  Follow these steps to set up your Windows computer to get started coding.

  For this you will need a *Windows 64-bit or 32-bit* computer, with an X64 processor, running Windows 11 (preferred) or Windows 10.

  Note: *If you are using a Windows computer with an ARM64 processor, you will need to follow the **Set up my Windows (with WSL)** setup guide.*

  <LinkCard
    title="Windows Software Setup Guide (using WSL)"
    href="/book/appendix/0-installation/2-6-setup-win-wsl/"
    description="Go to 'Set up my Windows (with WSL)' page"
  />
  <LinkCard
    title="Windows Software Setup Guide (using MSYS2)"
    href="/book/appendix/0-installation/2-5-setup-win-msys/"
    description="Go to 'Set up my Windows (with MSYS2)' page"
  />
</Card>

### <span style="display:none;">Linux</span>

<Card title="Linux" icon="fa-brands:linux">
  Follow these steps to set up your Linux computer to get started coding.

  For this you will need a *Linux* computer, or Virtual Machine, running the Debian distribution, or a distribution derived from Debian, such as Raspberry Pi OS.

  *Note: If you use a different Linux distribution, the steps may differ slightly.*

  <LinkCard
    title="Set up my Linux Box"
