title: Test your knowledge
---

Having explored this leg of the journey you are now approaching a rest stop and a chance to practice and reflect. Test what you have learnt by creating one or more of the following programs:

- [Simple Stats](/book/part-1-instructions/3-control-flow/4-activities/3-calculator) - a program that lets the user enter values and receive running statistics.
- [Simple music player](/book/part-1-instructions/3-control-flow/4-activities/4-music) - a program that lets the user load and play music.
- [Explore event loops](/book/part-1-instructions/3-control-flow/4-activities/5-events) - a program to let you explore event loops
