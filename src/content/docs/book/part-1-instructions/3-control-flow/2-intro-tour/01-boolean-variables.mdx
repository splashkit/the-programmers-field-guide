Boolean variables are used to hold one of two values: true or false.
These are very useful for conditions in our control flow statements.

Watch this video to:

- Understand the boolean data type.
- See how to create and use boolean variables.

When you have gone through this video you should be able to create and use boolean variables in your programs.
