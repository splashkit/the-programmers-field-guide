Learning to program is all about understanding how the code we create is run within the computer. The hand execution process is a great way to make sure _you_ know how the code runs. 

There are many ways that you could do this, but we want you to follow along with the approach we are showing here. This approach focuses on the actions the computer is performing, while other styles of hand execution (also called tracing) may focus more on determining the output or following an algorithm.

At the end of this you should have a start on how to do hand execution, and we will do more with this when we get to future chapters. It is something we will also include in our videos to help you understand what is happening.
