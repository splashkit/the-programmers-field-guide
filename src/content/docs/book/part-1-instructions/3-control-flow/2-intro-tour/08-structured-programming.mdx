Structured programming is based on the idea of arranging code as a series of blocks, where each block has a single entry and exit.
This helps us create programs with a clear sequence.

Watch this video to:

- Understand the purpose of structured programming.
- See how structured programs can be visualised.

At the end of this video you should be able to see how the programs you have been writing use the principles of structured programming.
