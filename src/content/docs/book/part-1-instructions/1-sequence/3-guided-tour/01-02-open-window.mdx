The first step in building a graphical program is to open a window to draw onto. In order to see the picture, we'll need to keep the window around for a few seconds before the program ends.

<Steps>
1. Add a call to open a window.

    In SplashKit you can open a window to draw onto. To open the window you call the `open_window` procedure, and pass it the windowâ€™s title, its width and its height.

    :::tip[Title and size?]
    You can give the window any title and size. Common sizes are usually in 4:3 such as 640x480, 800x600, 1024x768, or widescreen at 16:9 such as 1280x720, or 1920x1080.

    For this use the title "House Drawing by ..." and add your name, with a width of 800 and a height of 600.
    :::

2. If the program ends after the window is open, then it won't be around long enough for the user to see. To fix this we can use the `delay` procedure. You pass it an integer argument for the number of milliseconds to delay. Try delaying for 5000 milliseconds.

    <Accordion>
      <AccordionItem header="Code to open a window and delay">

      ```c++ {5-7}
      #include "splashkit.h"

      int main()
      {
          open_window("House Drawing by Andrew", 800, 600);

          delay(5000);
      }
      ```

      </AccordionItem>
    </Accordion>

3. Compile and run your program. You should see a window open. It should be rectangular, and have the title you set. After five seconds it should close.
</Steps>
