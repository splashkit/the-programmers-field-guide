Variables are used to store values that may change while your program is running. A good place to start with this is to use variables to accept input from the user.

Watch this video to:

- Get started declaring and using variables.
- See how to use functions to calculate values to store in variables.

When you are done, you should be able to create variables in your code and have a mental model of what is occurring.
