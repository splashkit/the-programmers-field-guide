Encapsulation is a principle for designing objects, where the fields and methods of an object can be *public* or *private*.
This allows us to prevent access to parts of the object which we want to keep hidden.

Watch this video to:

- Get started using access modifiers to design objects with the principle of encapsulation.

When you have gone through this video you should be able to design objects using access modifiers for fields and methods.
