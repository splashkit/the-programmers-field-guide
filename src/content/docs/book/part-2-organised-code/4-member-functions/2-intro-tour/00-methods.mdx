Methods are functions and procedures that are placed *inside* structs.
This allows us to store data and functionality together, to create entities which *know* and *can do* things.

Watch this video to:

- Get started with methods in structs.
- Understand the concept of an *object*.

When you have gone through this video you should be able to create and use arrays in your programs.
