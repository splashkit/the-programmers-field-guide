Arrays allow us to use a single variable to store multiple values of the same type.
This helps us to represent collections of related data.

Watch this video to:

- Get started with arrays.
- See how loops can be used to iterate through the elements of an array.

When you have gone through this video you should be able to create and use arrays in your programs.
