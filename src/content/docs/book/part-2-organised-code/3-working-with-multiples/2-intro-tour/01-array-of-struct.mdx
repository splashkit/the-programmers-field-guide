Arrays can store any type of data as its elements.
This includes primitive data, or structs.

Watch this video to:

- Get started with arrays of structs.
- See how the fields of a struct can be accessed through an array of structs.

When you have gone through this video you should be able to create and use arrays of structs in your programs.
