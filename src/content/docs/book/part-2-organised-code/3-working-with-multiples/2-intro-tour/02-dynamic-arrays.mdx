C++ arrays are not that nice to work with.
So, we have created fixed and dynamic array data types in SplashKit for you to use.

Watch this video to:

- Learn the difference between fixed and dynamic arrays.
- Get started with SplashKit fixed and dynamic arrays.
- See how to pass arrays into functions and procedures.

When you have gone through this video you should be able to create and use dynamic arrays in your programs.
