A common use case for arrays is to search for the "minimum" and "maximum" elements.
As programmers, we can define "minimum" and "maximum" however we want!
For example, in an array of numbers it could be the smallest and largest numbers.
Or, in an array of strings, it could be the shortest and longest strings.

Watch this video to:

- See how to design functions to find the minimum and maximum elements in an array.

When you have gone through this video you should be able to write `min` and `max` functions in your programs.
