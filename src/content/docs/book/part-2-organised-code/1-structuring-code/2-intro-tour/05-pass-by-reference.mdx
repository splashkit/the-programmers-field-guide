Pass by reference allows us to use parameters as normal variables.
Any parameter which is passed to a function or procedure by reference can be changed within that function or procedure.
Changes made to the parameter's value will persist even when the function or procedure has finished.

Watch this video to:

- Get started using pass by reference.
- See how the value of a variable can be changed within a function or procedure.
- Understand when to use pass by reference.

After this video you should be able to alter the values of variables within functions and procedures by using pass by reference.
