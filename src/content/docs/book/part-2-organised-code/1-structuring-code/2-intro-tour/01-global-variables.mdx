Global variables are variables which are accessible everywhere in a program.
While this sounds like a great idea, it quickly becomes a problem.

Watch this video to:

- Understand what a global variable is.
- See how global variables are dangerous to the integrity of our program data.

When you have gone through this video you should be able to understand the risks of using global variables, and why they should always be avoided.
