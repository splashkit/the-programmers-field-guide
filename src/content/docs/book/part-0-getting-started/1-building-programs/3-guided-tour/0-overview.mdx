title: Your Trip
---

Now it is your turn to get your computer ready to for you to build and run programs. This will involve:

- Installing the tools you need to work through the exercises and activities in this guide.
- Use these tools to build and run programs we give you.
- Experience the general programming workflow, the steps you will perform when you start building your own programs.

By the end of these activities you should feel confident that you can:

- Write Create code using Visual Studio Code (VS Code).
- Build and run programs using the Terminal in VS Code.
- Describe the different part of the commands you run in the terminal.
- Describe and use the process for iteratively building and editing programs.

:::tip
Make sure you are actively working through this material. The skills you are aiming to learn will require practice. It is like learning to juggle or ride a bike, you can read about how to do this and watch videos of others doing it, but to really learn these skills you need to get active and have a go yourself.

To actively work through the material, turn off your notifications, so you can focus, and as you read through a section make sure to perform the actions descried yourself.
:::
