[SplashKit](https://splashkit.io) is a library we have created to help supercharge the programs you write as you learn how to program. This library will allow us to focus on the concepts, and transferrable skills that will set you up to succeed in programming across a range of languages and devices.

Watch the video to:

- Get some insights into SplashKit.
- Understand its role in supporting your learning.

By making the most of SplashKit, you should be able to create fun and engaging programs while you are learning.
